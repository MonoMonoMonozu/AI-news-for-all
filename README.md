# 📰 みんなのAIニュース — Echo / Sonar / Lucky

**"探知したよ📡 今朝のAIトレンドをお届け。"**

`みんなのAIニュース` は、Anthropic Claude APIのリアルタイム検索機能を活用し、膨大なAIニュースを3つの視点（チャンネル）でフィルタリングして届ける、対話型ニュースキュレーションアプリです。[cite: 2]

---

## 🌟 コンセプト：AIキャスターとの対話

このアプリは単なるニュース一覧ではありません。3人のAIキャラクターが、あなたの「相方」としてニュースを解説します。[cite: 2]

*   **📰 経済新聞 (Echo 🦭):** ビジネス・経済視点。[cite: 2]
*   **💻 ユーザーボイス (Lucky 🦝):** エンジニア向け。実用的・具体的な技術動向。[cite: 2]
*   **👀 御三家動向 (Sonar 🐬):** OpenAI / Anthropic / Googleの動向に特化。[cite: 2]

---

## ✨ 主な機能

*   **リアルタイム探知:** `web_search` を使用し、今日の最新ニュースを自動収集。[cite: 2]
*   **マルチリージョン:** 🌍 ワールドと 🇯🇵 日本の切り替えに対応。[cite: 2]
*   **深掘りチャット:** ニュースの背景・意義・ポイントをAIが即座に解説。[cite: 2]
*   **取材ポリシー表示:** AIがどのように情報を集めているかを透明化。[cite: 2]

## 🚀 クイックスタート
